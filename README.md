# Fuliza Game

## Project Overview
Fuliza Game is an engaging and interactive game that allows users to experience fun and challenging gameplay.

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/dgichiri/fulizagame.git
   ```
2. Navigate to the project directory:
   ```
   cd fulizagame
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Deployment Guide
1. Build the project:
   ```
   npm run build
   ```
2. Deploy the project:
   Follow your hosting provider's instructions to deploy the contents of the `dist/` directory.
